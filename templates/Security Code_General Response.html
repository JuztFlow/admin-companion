In order to reset, change or disable the security code of your account, please follow these steps:
  1. Important: Make sure that you have the game closed, or at least be logged out of your ingame account, before proceeding, because it will definitely NOT work if your account is still logged ingame while doing the next steps. This is a very common mistake that happens to a lot of players, so please be aware in order to avoid it!

  2. Now, after you made sure that you have the game closed or are at least logged out of your ingame account, go to the "Account Management" page on our website (https://vision4s.com/manage) and make sure that you are logged on the website with the correct account that you want to reset/change/disable the security code of.

  3. Once you are sure that you are logged in with the correct account, click on the "RESET SECURITY CODE" button on the left side of the page or alternatively, follow this link: https://vision4s.com/manage/security

  4. You will be asked to solve a captcha in order to proceed. After you done that, just click on the "SEND" button.

  5. Now you will receive an email from 'noreply@vision4s.com' with further instructions and a link to confirm the reset your security code, which you need to click on. In the email, it reads: "CLICK HERE TO RESET SECURITY CODE". Please also make sure to check your spam folder in case you don't see the email in your inbox.

  6. After you clicked on the link "CLICK HERE TO RESET SECURITY CODE" in the email in order to confirm the reset, you then will receive a second email from 'noreply@vision4s.com' which finally contains your newly issued, and therefore random, security code.

  7. Only now, you can log back into the game and use the ingame functions in order to change/disable your security code using the new one you received. If you logged back into the game before you received the second email with the new security code, you will have to repeat the whole process again because it will always be invalid.

Please let us know if you still have any trouble with the process.
